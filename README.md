## 🇺🇸 English Introduction

### Overview
CasualStudy is a modern, responsive personal portfolio and data visualization hub. Built with a stunning **Glassmorphism UI**, the platform aggregates various quantitative finance studies, market dashboards, and AI industry trackers. It features seamless bilingual (English & Chinese) switching, mobile-first design, and interactive ECharts data visualizations.

### 🌟 Key Features & Dashboards

1. **AI Revenue & Token Usage Tracker** (`ai-revenue-tracker.html`)
   - Tracks daily/weekly revenue and token usage for the Top 20 LLM AI models (e.g., GPT-4, Claude 3.5, Gemini).
   - Interactive model selectors and dynamic timeframe aggregation.
2. **Institutional Fund Holdings (13F)** (`fund-holdings.html`)
   - Visualizes quarterly portfolio changes for top quantitative hedge funds (e.g., Renaissance Technologies, Two Sigma, Citadel).
3. **South Korea Margin Loan Balance Analysis** (`korea-margin.html`)
   - Tracks the dynamic balance of retail margin debt in the Korean stock market and its correlation with KOSPI/KOSDAQ.
4. **CNN Fear & Greed Index Tracker** (`fear-greed.html`)
   - Historical tracking and sentiment analysis dashboard based on the CNN Fear & Greed Index.
5. **CFM Market Neutral Tracker** (`cfm-market.html`)
   - Monitors Equity Market Neutral strategies and relevant index performances.
6. **Market Gap Study** (`gap-study.html`)
   - Statistical analysis of overnight gaps vs. intraday returns in major indices.
7. **Index Inclusion Effect Analysis** (`inclusion-study.html`)
   - Research on the price impact before and after a stock is added to or removed from a major market index.
8. **Quant Admin Dashboard** (`quant-admin.html`)
   - A unified administrative interface for managing quantitative scripts and data pipelines.

### 🛠 Technology Stack
- **Frontend**: HTML5, CSS3 (Custom Glassmorphism), Vanilla JavaScript.
- **Visualizations**: [Apache ECharts](https://echarts.apache.org/) for interactive, responsive, and mobile-optimized charting.
- **Design Patterns**: Mobile-first responsiveness, Horizontal Carousel Selectors, Dark/Light Mode Auto-adaptation.
- **Deployment**: GitHub Pages via automated Python/cron data pipelines (VPS).

## 🇨🇳 中文简介

### 项目概述
CasualStudy 是一个现代化的个人展示与数据可视化平台。网站采用了极具美感的 **毛玻璃 (Glassmorphism)** 设计语言，聚合了大量的量化金融研究成果、市场情绪仪表盘以及人工智能行业的动态追踪。全站支持中英文无缝切换，深度适配移动端操作，并基于 ECharts 提供了丝滑的数据交互体验。

### 🌟 核心功能与页面详解

1. **AI 大模型流水与调用量追踪** (`ai-revenue-tracker.html`)
   - 追踪全网 Top 20 核心 AI 模型（如 GPT-4, Claude 3.5, Gemini 等）的每日/每周预估流水和 Token 消耗量。
   - 移动端定制的胶囊按钮（Pills）交互，轻松切换对比特定模型的市场份额。
2. **机构基金持仓追踪 (13F)** (`fund-holdings.html`)
   - 深入分析顶级量化对冲基金（如文艺复兴、Two Sigma、Citadel 等）的季度调仓行为与资产配置比例。
3. **韩国市场融资融券余额分析** (`korea-margin.html`)
   - 追踪韩国散户融资买入余额（Margin Loan Balance）的动态变化，及其与 KOSPI/KOSDAQ 指数的历史相关性。
4. **CNN 恐惧与贪婪指数监控** (`fear-greed.html`)
   - 可视化展示市场情绪的历史走向与极端情况分布。
5. **CFM 市场中性策略追踪** (`cfm-market.html`)
   - 监控股票市场中性 (Equity Market Neutral) 相关指数的表现与统计特征。
6. **市场跳空缺口研究** (`gap-study.html`)
   - 针对核心指数的隔夜跳空幅度与日内收益率之间的统计学联系分析。
7. **指数纳入效应研究** (`inclusion-study.html`)
   - 追踪和回测股票在被主流指数纳入或剔除前后的异常收益率（Abnormal Returns）表现。
8. **量化后台管理** (`quant-admin.html`)
   - 个人量化脚本、数据抓取任务（Cron Jobs）与 VPS 服务器运行状态的统一管理看板。

### 🛠 技术栈
- **前端核心**: HTML5, CSS3 (深度定制的毛玻璃风格), 原生 JavaScript。
- **数据可视化**: [Apache ECharts](https://echarts.apache.org/) - 实现复杂图表在手机与电脑端的高性能渲染与触控优化。
- **设计规范**: 移动端优先适配 (Mobile-first)、横向滑动式胶囊选择器、明暗模式 (Dark/Light Mode) 自动适应。
- **后端/部署**: 基于 GitHub Pages 托管，数据由独立 VPS 上的 Python 定时任务生成并推送。
